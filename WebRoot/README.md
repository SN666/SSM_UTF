#Minimal-Admin

### 基于ZUI进行二次开发定制的后台模板

项目中采用requie.js进行模块加载  
项目采用模板页架构，只有一个主页html，其他的每个功能子页均采用js渲染，通过requie.js进行加载  
项目中html 模板采用doT.js ，提高html在js 中的可读性
